# ShogiProject
## 概要
将棋の棋譜の管理を管理するシステムです。
AIによる局面の解析機能も有しています。

このシステムは本リポジトリと
[ShogiProject_Analysis](https://github.com/h-akira/ShogiProject_Analysis)
2つのリポジトリからなります。
また、フレームワークとして
[hads](https://github.com/h-akira/hads)
を使用しています。

## 構成
